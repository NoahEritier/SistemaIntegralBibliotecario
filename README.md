# Códice - Plataforma Integral de Servicios Bibliotecarios (LSP)

Plataforma moderna y elegante para la gestión integral de servicios bibliotecarios, diseñada con una estética "Modern Intellectual" minimalista y sofisticada.

## Stack Tecnológico

- **Frontend**: React 18 + Vite + TypeScript
- **Estilos**: Tailwind CSS (configuración estricta)
- **Iconos**: Lucide React (solo iconos de línea fina)
- **Estado**: Zustand
- **Routing**: React Router DOM
- **Gráficos**: Recharts (estilizados minimalistas)
- **Utilidades**: clsx + tailwind-merge

## Paleta de Colores

- **Fondo Principal**: `#F9F7F2` (Crema pálido/Papel antiguo limpio)
- **Fondo Secundario/Tarjetas**: `#FFFFFF` (Blanco puro)
- **Texto Principal**: `#1A1A1A` (Casi negro)
- **Texto Secundario**: `#5C4B45` (Marrón tierra desaturado)
- **Acentos/Bordes**: `#8C7B75` (Taupe/Marrón grisáceo)
- **Bordes Activos**: `#2C1E1A` (Marrón café muy oscuro)
- **Error**: `#943E3E` (Rojo óxido apagado)
- **Éxito**: `#4A5D44` (Verde bosque desaturado)

## Tipografía

- **Títulos**: Playfair Display (Serif, elegante)
- **Cuerpo/Datos**: Inter (Sans-serif, legible)

## Estructura del Proyecto

```
src/
├── components/
│   └── ui/          # Componentes UI reutilizables
├── features/        # Features modulares
├── layouts/         # Layouts de la aplicación
├── pages/           # Páginas principales
├── router/          # Configuración de rutas
├── store/           # Stores de Zustand
└── utils/           # Utilidades (cn, helpers, etc.)
```

## Principios de Diseño

1. **Minimalismo Sofisticado**: Espaciado generoso, elementos limpios
2. **Bordes Finos**: 1px, border-radius pequeño (4px-6px)
3. **Sin Sombras**: Preferir bordes a sombras difusas
4. **Tipografía Elegante**: Serif para títulos, sans-serif para contenido
5. **Colores Restringidos**: Solo usar la paleta definida
